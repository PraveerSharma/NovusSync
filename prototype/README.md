# Gate G0 Clickable Prototype

This is a disposable, dependency-free owner-test artifact. It is not the application scaffold.

## Open it

Open `prototype/index.html` in a modern browser. No install, package manager, account, environment variable, or network connection is required.

For a local HTTP URL, from the repository root run:

```sh
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000/prototype/`.

## Test it

Give an eligible owner the five tasks without explaining the interface:

1. Correct the introductory-offer price using the source evidence.
2. Approve the changed carousel's exact current version.
3. Explain who publishes Instagram content and sends WhatsApp messages.
4. Record the requested opt-out for synthetic Lead L-103, then explain the difference between owner takeover and campaign pause.
5. Verify the attended outcome and accept, reject, or defer the proposed experiment. You may also inspect how no-show, cancellation, and superseding corrections remain separate from attendance.

Use the coded test record in [the prototype plan](../docs/discovery/CLICKABLE_PROTOTYPE_PLAN.md). Record every wrong turn, question, prompt, objection, and missing step before helping.

## Safety boundary

- All names, accounts, references, dates, and numbers are fictional or synthetic.
- No data is saved between reloads.
- No request leaves the browser.
- No Instagram, WhatsApp, booking, payment, AI, analytics, authentication, or provider integration exists.
- The interface cannot prove Gate G0. Five eligible-owner tests and the recorded evidence are still required.
